# Manual setup

## Homebrew and Apps
```
brew install mas
brew install rcmdnk/file/brew-file
brew install mackup
curl -sf https://raw.githubusercontent.com/yamatsum/mackup/master/setup.sh | sh -s
mackup restore
mas signin <apple ID>
brew file install
```
